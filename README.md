## 🚀 Bắt đầu

1. Đọc [`docs/00-setup-environment.md`](./docs/00-setup-environment.md) để cài môi trường
2. Đọc doc lý thuyết tương ứng với từng project
3. Làm từng project theo thứ tự **01 → 05**
4. Mỗi project có `README.md` riêng hướng dẫn chi tiết
