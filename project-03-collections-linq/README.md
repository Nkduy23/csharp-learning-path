# 🟡 Project 03 — Collections & LINQ

**Độ khó:** ⭐⭐⭐  
**Chủ đề:** List, Dictionary, HashSet, LINQ, Lambda, GroupBy, Join

---

## 🎯 Mục tiêu

Xây dựng **E-commerce Product Catalog** — hệ thống quản lý sản phẩm với LINQ queries mạnh.

---

## 📋 Chức năng cần làm

- [ ] CRUD sản phẩm (tên, giá, danh mục, stock)
- [ ] Tìm kiếm & lọc sản phẩm (theo giá, danh mục, tên)
- [ ] Sắp xếp nhiều tiêu chí (ThenBy, ThenByDescending)
- [ ] Thống kê theo danh mục (GroupBy)
- [ ] Giỏ hàng — thêm/xóa/tính tổng (Dictionary)
- [ ] Top 5 sản phẩm bán chạy
- [ ] Áp dụng voucher / discount (LINQ + filter)

---

## 💡 Kiến thức áp dụng

- LINQ: `Where`, `Select`, `OrderBy`, `GroupBy`, `Join`
- `Dictionary<string, List<Product>>`
- `HashSet<int>` cho wishlist (không duplicate)
- Lambda expressions & method chaining
- Anonymous types `new { Name, Price }`
- Deferred execution
